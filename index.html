🔗 GitHub Gist Sync

Save and load your opportunity edits to a private GitHub Gist so they sync across browsers and devices. Your token stays in this browser only — it is never sent anywhere except api.github.com.

Create at github.com/settings/tokens · scope needed: gist only.
If blank, the first Push will create a new secret Gist named 4sight-opportunities.json and remember its ID here.
How it works: Click Push to upload your saved opportunities to a secret Gist. On another machine, paste the same token + Gist ID and click Pull to load them. Turn on Auto-sync to keep them updated automatically as you edit.
